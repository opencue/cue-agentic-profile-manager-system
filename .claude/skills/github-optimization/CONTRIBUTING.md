# Contributing

Thanks for your interest in making this skill better.

## 3-Step Process

1. **Fork** this repo
2. **Edit** -- improve the skill, fix a bug, add a pattern that drives more stars
3. **Open a PR** with a clear description of what changed and why

## What We're Looking For

- README patterns that convert visitors into stargazers
- Better badge configurations or CTA placements
- Writing rules that catch more AI slop
- SEO techniques that improve GitHub search ranking
- Bug fixes or edge cases the skill doesn't handle

## Guidelines

- Keep the voice direct and human -- no AI slop in PRs either
- Test your changes on a real repo before submitting
- One PR per change -- don't bundle unrelated edits
