The shortest possible path from "I have soul installed" to "claude is loaded
with only the skills I need for the current task."

```bash
soul list                       # see what profiles exist
soul use medusa-dev --here      # materialize medusa-dev in this directory
claude                          # launches with .claude/skills/ + .mcp.json picked up
```
