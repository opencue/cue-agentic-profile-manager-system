# medusa-dev

Medusa Dev is tuned for the user's shop work: backend modules, workflows, storefront API calls, admin widgets, migrations, and repeatable shop setup. It inherits core so commit discipline and repo-reading stay loaded without pulling every Medusa-adjacent utility.
